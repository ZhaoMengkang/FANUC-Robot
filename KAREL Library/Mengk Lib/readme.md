# MK_LIB – FANUC KAREL Utility Library

**Version:** V25.12
**Author:** Zhao, Mengkang
**Date:** 2025-12-07

---

## 📌 Overview

`MK_LIB` is a comprehensive **FANUC KAREL utility library** designed to simplify:

* Data type conversion (INT, REAL, BYTE, STRING)
* Bitwise operations
* IO (DIN/DOUT) data handling
* Socket communication (Client/Server)
* String processing utilities
* Modbus CRC calculation
* TP parameter access

The library is optimized for **robust industrial robot communication and data processing**.

## 🔧 Function Reference

---

## 🔢 Conversion Functions

### `CNV_INT2STR`

Convert INTEGER to STRING (removes leading blank generated by system conversion).

| Parameter | Type    | Direction | Description              |
| --------- | ------- | --------- | ------------------------ |
| in_int    | INTEGER | IN        | Integer value to convert |

**Returns:** STRING (trimmed result)

---

### `CNV_REL2STR`

Convert REAL to STRING (removes leading blank).

| Parameter | Type | Direction | Description                   |
| --------- | ---- | --------- | ----------------------------- |
| in_real   | REAL | IN        | Real (float) value to convert |

**Returns:** STRING

---

### `CNV_INT2HEX`

Convert a non-negative integer into uppercase hexadecimal string.

| Parameter | Type    | Direction | Description                         |
| --------- | ------- | --------- | ----------------------------------- |
| val       | INTEGER | IN        | Input decimal value (≥ 0)           |
| out_str   | STRING  | OUT       | Output hexadecimal string (no `0x`) |

---

## 🔢 Bit / Byte Operations

### `POW2`

Compute power of 2.

| Parameter | Type    | Direction | Description      |
| --------- | ------- | --------- | ---------------- |
| n         | INTEGER | IN        | Exponent (n ≥ 0) |

**Returns:** INTEGER (2ⁿ)

---

### `CNV_BYTE2BIT`

Convert BYTE (0–255) into bit array.

| Parameter | Type                | Direction | Description                                |
| --------- | ------------------- | --------- | ------------------------------------------ |
| val       | INTEGER             | IN        | Byte value (0–255)                         |
| out_bits  | ARRAY[8] OF INTEGER | OUT       | Bit array (MSB at index 1, LSB at index 8) |

---

### `CNV_BIT2BYTE`

Convert bit array into BYTE.

| Parameter | Type                | Direction | Description           |
| --------- | ------------------- | --------- | --------------------- |
| in_bits   | ARRAY[8] OF INTEGER | IN        | Bit array (MSB first) |
| val       | INTEGER             | OUT       | Output byte value     |

---

### `SHL`

Arithmetic left shift.

| Parameter | Type    | Direction | Description             |
| --------- | ------- | --------- | ----------------------- |
| val       | INTEGER | IN        | Input value             |
| n         | INTEGER | IN        | Number of bits to shift |

**Returns:** INTEGER

---

### `SHR`

Arithmetic right shift.

| Parameter | Type    | Direction | Description             |
| --------- | ------- | --------- | ----------------------- |
| val       | INTEGER | IN        | Input value             |
| n         | INTEGER | IN        | Number of bits to shift |

**Returns:** INTEGER

---

### `XOR_FUNC`

Bitwise XOR operation.

| Parameter | Type    | Direction | Description |
| --------- | ------- | --------- | ----------- |
| a         | INTEGER | IN        | Operand A   |
| b         | INTEGER | IN        | Operand B   |

**Returns:** INTEGER (A XOR B)

---

## 🔄 REAL ↔ BYTE (IEEE754)

### `CNV_REL2BYTE`

Convert REAL to IEEE754 byte array.

| Parameter | Type             | Direction | Description          |
| --------- | ---------------- | --------- | -------------------- |
| r_val     | REAL             | IN        | Input float value    |
| is_little | BOOLEAN          | IN        | TRUE = little-endian |
| byte_arr  | ARRAY[4] OF BYTE | OUT       | Output byte array    |

---

### `CNV_BYTE2REL`

Convert IEEE754 byte array to REAL.

| Parameter | Type             | Direction | Description      |
| --------- | ---------------- | --------- | ---------------- |
| byte_arr  | ARRAY[4] OF BYTE | IN        | Input byte array |
| is_little | BOOLEAN          | IN        | Endianness flag  |
| r_val     | REAL             | OUT       | Output float     |

---

## 🔌 IO (Digital Input/Output)

### `GI32`

Read 32 digital inputs and convert to REAL.

| Parameter | Type    | Direction | Description                     |
| --------- | ------- | --------- | ------------------------------- |
| io_start  | INTEGER | IN        | Start DI index                  |
| io_end    | INTEGER | IN        | End DI index (must be start+31) |
| is_little | BOOLEAN | IN        | Endianness                      |
| real_val  | REAL    | OUT       | Converted float value           |

---

### `GO32`

Write REAL value to 32 digital outputs.

| Parameter | Type    | Direction | Description     |
| --------- | ------- | --------- | --------------- |
| io_start  | INTEGER | IN        | Start DO index  |
| io_end    | INTEGER | IN        | End DO index    |
| is_little | BOOLEAN | IN        | Endianness      |
| real_val  | REAL    | IN        | Value to output |

---

## 🌐 Socket Communication

### Supported Tags

* Client: `C1:` ~ `C8:`
* Server: `S1:` ~ `S8:`

---

### `SM_CONFIG`

Configure socket parameters automatically.

| Parameter | Type    | Description             |
| --------- | ------- | ----------------------- |
| tag_nam   | STRING  | Socket tag (e.g. `C2:`) |
| ip_addr   | STRING  | Remote IP address       |
| port      | INTEGER | Port number             |

⚠ Requires controller **cold start**

---

### `SM_CONNECT`

Connect socket after ping validation.

| Parameter | Type   | Description |
| --------- | ------ | ----------- |
| tag_nam   | STRING | Socket tag  |

---

### `SM_DISCON`

Disconnect socket.

| Parameter | Type   | Description |
| --------- | ------ | ----------- |
| tag_nam   | STRING | Socket tag  |

---

### `SM_SEND`

Send data via socket.

| Parameter | Type   | Direction | Description  |
| --------- | ------ | --------- | ------------ |
| tag_nam   | STRING | IN        | Socket tag   |
| sm_file   | FILE   | IN        | File handle  |
| data      | STRING | IN        | Data to send |

**Returns:** INTEGER (status code)

---

### `SM_RECV`

Receive data from socket.

| Parameter   | Type    | Direction | Description                |
| ----------- | ------- | --------- | -------------------------- |
| tag_nam     | STRING  | IN        | Socket tag                 |
| sm_file     | FILE    | IN        | File handle                |
| data        | STRING  | OUT       | Received data              |
| remain_data | INTEGER | OUT       | Remaining unread bytes     |
| timeout     | INTEGER | IN        | Timeout (ms, 0 = infinite) |

---

### `SM_PING`

Ping remote device.

| Parameter | Type   | Description |
| --------- | ------ | ----------- |
| tag_nam   | STRING | Socket tag  |

**Returns:** INTEGER (0 = success)

---

## 🧮 Modbus

### `MODBUS_CRC16`

Calculate Modbus RTU CRC16.

| Parameter | Type          | Direction | Description    |
| --------- | ------------- | --------- | -------------- |
| data      | ARRAY OF BYTE | IN        | Input frame    |
| len       | INTEGER       | IN        | Length of data |
| crc       | INTEGER       | OUT       | CRC result     |

---

## 🔤 String Utilities

### `E_SPLIT`

Split string by delimiter.

| Parameter    | Type            | Direction | Description          |
| ------------ | --------------- | --------- | -------------------- |
| in_str       | STRING          | IN        | Source string        |
| split_str    | STRING          | IN        | Delimiter (1 char)   |
| no_ctrl_code | BOOLEAN         | IN        | Ignore control chars |
| out          | ARRAY OF STRING | OUT       | Result array         |
| count        | INTEGER         | OUT       | Segment count        |

**Returns:** INTEGER (0 = OK)

---

### `E_SPLIT_ARR`

Split array of strings.

| Parameter    | Type            | Direction | Description        |
| ------------ | --------------- | --------- | ------------------ |
| in_arr       | ARRAY OF STRING | IN        | Input array        |
| arr_len      | INTEGER         | IN        | Number of elements |
| split_str    | STRING          | IN        | Delimiter          |
| no_ctrl_code | BOOLEAN         | IN        | Skip control chars |
| out_arr      | ARRAY OF STRING | OUT       | Output segments    |
| count        | INTEGER         | OUT       | Total segments     |

---

### `REPLACE`

Replace substring occurrences.

| Parameter | Type   | Direction | Description      |
| --------- | ------ | --------- | ---------------- |
| in_str    | STRING | IN        | Source string    |
| old_pat   | STRING | IN        | Target substring |
| new_pat   | STRING | IN        | Replacement      |
| out_str   | STRING | OUT       | Result string    |

---

### `TO_UPPER`

Convert lowercase letters to uppercase.

| Parameter | Type   | Direction | Description      |
| --------- | ------ | --------- | ---------------- |
| in_str    | STRING | IN        | Input string     |
| out_str   | STRING | OUT       | Uppercase result |

---

### `TO_LOWER`

Convert uppercase letters to lowercase.

| Parameter | Type   | Direction | Description      |
| --------- | ------ | --------- | ---------------- |
| in_str    | STRING | IN        | Input string     |
| out_str   | STRING | OUT       | Lowercase result |

---

### `EXTR_BLOCK`

Extract substring between tags (supports nested structure).

| Parameter | Type   | Direction | Description       |
| --------- | ------ | --------- | ----------------- |
| src_str   | STRING | IN        | Source string     |
| start_tag | STRING | IN        | Start tag         |
| end_tag   | STRING | IN        | End tag           |
| content   | STRING | OUT       | Extracted content |

**Returns:** INTEGER (0 = OK)

---

### `EXTR_B_ARR`

Extract block from string array (multi-line support).

| Parameter | Type            | Direction | Description  |
| --------- | --------------- | --------- | ------------ |
| in_arr    | ARRAY OF STRING | IN        | Input lines  |
| arr_len   | INTEGER         | IN        | Line count   |
| start_tag | STRING          | IN        | Start tag    |
| end_tag   | STRING          | IN        | End tag      |
| out_arr   | ARRAY OF STRING | OUT       | Output lines |
| out_cnt   | INTEGER         | OUT       | Output count |
